Screen Time vs Mental Wellness — Final Project

Introduction

This project investigates how daily screen exposure influences mental wellness indicators such as sleep quality, stress levels, mood, and energy. Using a fictional dataset of 400 users, we analyze patterns, explore correlations, and generate insights to inform healthier digital habits.

Methodology

The dataset includes demographics, device-specific screen time (mobile, laptop, TV), total daily hours, sleep quality ratings, stress levels, productivity scores, and mood indicators. We performed exploratory data analysis, trend visualization, and correlation heatmapping using Plotly.js.

Insights

Higher screen exposure correlates with increased stress and poorer sleep quality. Three user groups emerge: balanced users, high-screen professional users, and at-risk users with 7–9 hours of daily screen time. Recommendations include digital cut-off windows, app-based nudges, and device-free zones.
